CrystalsCollector

Instructions:

When the game begins, a random number is selected, and each crystal is given a random value. Your job is to match the random number by clicking on the crystals. You will not see the value of each crystal, but you can identify their values by clicking on each crystal.

The game ends when your score either matches or exceeds that of the random number. When the game ends, a new value will be given to each crystal, and a new random number will be selected. Your score is recorded in the aqua colored box below.
